# 火一五·辉火云企业套件插件

> OpenClaw 插件 — 自然语言操作辉火云企业套件（Odoo 19）

## 功能

- 🤖 自然语言操作（待办、日历、CRM商机、销售订单等）
- 🔔 通知同步（待办、活动、消息）
- ⚡ 30+ 意图模式

## 安装

```bash
openclaw plugins install @huo15/openclaw-odoo-dev
```

## 工具

- odoo_connect - 连接 Odoo
- odoo_create_task - 创建待办
- odoo_list_tasks - 待办列表
- odoo_create_event - 创建日历
- odoo_status - 连接状态
